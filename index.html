LifeBench Data Pipeline

생성형 AI 기반 개인 생활 데이터 시뮬레이션 파이프라인

⏱️ 전체 실행 시간: ~2-4시간 (1년치 데이터)
👤

Step 1

Persona

인물 프로필 생성

나이, 직업, 성격(MBTI), 연봉 등 기본 인물 정보 생성
📝

Step 2

Draft Generator

연간 타임라인 초안

1년간 주요 생활 이벤트 타임라인 생성 (일별 이벤트 목록)
🎮

Step 3

Simulator

상세 이벤트 시뮬레이션

일별 상세 이벤트 + 건강 상태 추적 (신체/정신/수면/에너지)
📱

Step 4

Phone Data

스마트폰 데이터 생성

전화, SMS, 푸시알림, 캘린더, 메모, 사진, 피트니스 데이터 생성
❓

Step 5

QA Generator

질의응답 데이터 생성

단일/다중 홉 QA, 추론 문제 생성 (생성된 데이터 기반)

수정됨
