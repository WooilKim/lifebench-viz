핵심 과제: 지정 월의 중요 이벤트와 변화 기록 분석, 이벤트의 영향 유형 식별 (고정 변화와 근접 영향)

## 과제 요구사항:

### 1. 월간 이벤트 분석
- 지정 월의 중요 이벤트 선별·분석 (직업, 생활, 학습, 건강, 사교 등 각 카테고리)
- 월간 관련 카테고리 요약과 결합하여 이벤트 최적화 분석 (이벤트 추가·삭제·수정 가능)
  - 이벤트 발생 날짜 등 속성 조정으로 불합리한 충돌 이벤트 삭제
  - 다양성·연속성 이벤트 추가로 개인의 한 달 중요 이벤트와 변화를 기록
- 이벤트 설명 정확하고 완전하며 타임라인 명확하게 보장

### 2. 영향 분석
- **고정 변화**: 이벤트가 초래하는 장기적 정보 변화 식별. 이후 모든 이벤트가 이와 충돌하는지 고려해야 함.
  - 예: XX 동료 퇴직, 이후 업무에서 협업하지 않음, 동료에서 친구로 관계 변화
  - 구체적인 변화 내용을 명확히 지적해야 함
- **근접 영향**: 이벤트가 초래하는 단기적 생활 상태 변화 식별 (특정 기간)
  - 예: 출장 1주일 (XX일~XX일), 이 기간 집에 없어서 자주 가는 XX 피트니스센터를 갈 수 없음, XX 회의 참석을 위해 일찍 기상 필요
  - 영향 기간과 구체적인 상태 변화를 명확히 해야 함

### 3. 출력 형식 요구사항

엄격한 JSON 형식으로 월간 이벤트 분석 결과를 출력하세요:

{
  "month": "[월, 형식: YYYY-MM]",
  "month_overview": "[월 전체적인 발전 개요]",
  "key_events": [
    {
      "time_point": "[구체적 날짜, 형식: YYYY-MM-DD]",
      "event": "[이벤트 설명]",
      "fixed_changes": [
        "[고정 변화1 설명]",
        "[고정 변화2 설명]"
      ],
      "short_term_impacts": [
        {
          "start_date": "[영향 시작 날짜, 형식: YYYY-MM-DD]",
          "end_date": "[영향 종료 날짜, 형식: YYYY-MM-DD]",
          "impact_description": "[근접 영향 설명]"
        }
      ]
    }
  ]
}

## 입력 데이터:
### 인물 프로파일:
{character_profile}

### 지정 월:
{target_month}

### 해당 월의 이벤트 목록:
{monthly_events}

### 카테고리별 연간 이벤트 요약:
{category_summaries}
