핸드폰 데이터 생성 전문가로서, 다음 이벤트 정보를 분석하고 해당 이벤트를 반영하기 위해 생성해야 할 핸드폰 조작 데이터 유형(1-3개)을 판단하세요.

## 입력 정보
- 날짜: {date}
- 당일 이벤트 배경: {daily_events_str}
- 대상 이벤트 정보: {event_info}

## 선택 가능한 데이터 유형
1. 문자 (sms)
2. 통화 (call)
3. 사진 (photo)
4. 알림 (notification)
5. 메모 (note)
6. 캘린더 (calendar)

## 분석 요구사항
각 데이터 유형에 대해 다음을 포함:
- data_type: 데이터 유형
- reason: 이 유형을 선택한 이유 (이벤트와의 연관성)
- key_content: 생성할 핵심 내용 요약
- participants: 관련 참여자 (연락처 기반)

출력: JSON 배열 형식
